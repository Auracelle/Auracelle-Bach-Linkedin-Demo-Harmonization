# Quick Start Guide - Auracelle Bach

## Getting Your Demo Live in 5 Minutes

### Step 1: Create Repository
1. Go to [github.com](https://github.com) and log in
2. Click the green "New" button
3. Name it: `Auracelle-Bach`
4. Select "Public"
5. Do NOT check "Add a README file"
6. Click "Create repository"

### Step 2: Upload Files
1. Click "uploading an existing file"
2. Drag and drop all files:
   - `index.html`
   - `README.md`
   - `LICENSE`
   - `DEPLOYMENT.md`
   - `QUICKSTART.md`
3. Add commit message: "Initial Bach demo upload"
4. Click "Commit changes"

### Step 3: Enable GitHub Pages
1. Click "Settings" in your repository
2. Click "Pages" in the left menu
3. Under "Source," select "Deploy from a branch"
4. Under "Branch," select "main"
5. Under folder, select "/ (root)"
6. Click "Save"
7. Wait 1-2 minutes and refresh

## Updating Your Demo

To update the demo later:
1. Go to your repository
2. Click on `index.html`
3. Click the pencil icon (edit)
4. Make changes
5. Commit changes
6. Site updates automatically in 1-2 minutes!
